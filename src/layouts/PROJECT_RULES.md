# Project Rules

## Goal
Build a fast, beautiful, SEO-first micro-tool website.

## Design
- Mobile-first.
- Clean modern UI.
- Minimalist.
- Fast loading.
- Dark mode support.
- Accessible.

## Code
- Keep components small.
- Reuse components whenever possible.
- Avoid unnecessary dependencies.
- Use Astro best practices.

## Performance
- Optimize for Core Web Vitals.
- Lazy load images.
- Keep JavaScript minimal.

## SEO
- Every page must have:
  - Title
  - Meta description
  - Canonical URL
  - Open Graph tags
  - Twitter Card tags

## Pages
Every website should eventually include:
- Home
- About
- Contact
- Privacy Policy
- Terms of Service

## Content
- Human-friendly.
- No keyword stuffing.
- Easy to read.

## General
Always explain major architectural decisions before implementing large changes.
